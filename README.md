💰 Sistema de Controle Financeiro

Sistema desktop desenvolvido em Python com PyQt para gerenciamento financeiro completo, com arquitetura modular e recursos avançados.

---

🚀 Visão Geral

O sistema permite o controle completo de finanças pessoais ou operacionais, incluindo contas, cartões de crédito, transações e planejamento financeiro.

Projetado com foco em:

- organização financeira
- performance
- interface moderna
- escalabilidade

---

⚙️ Funcionalidades

- 📊 Controle de contas bancárias
- 💳 Gestão de cartões de crédito
- 🔁 Transferência entre contas
- 📥 Importação inteligente de extratos
- 📤 Exportação de dados (CSV, XLSX, PDF)
- 🎯 Metas financeiras
- 📅 Agendamentos financeiros
- 🎨 Tema claro/escuro
- 🧩 Arquitetura modular (MVC)

---

🖥️ Tecnologias

- Python
- PyQt5
- SQLite
- SVG Icons
